How to Join NITK SMP

Note: We use Prism Launcher (Open Source & Trustworthy). ⚠️ Do not trust Tlauncher.

Installation Steps

  1. Download Prism Launcher (v9.4) from the official link:
    Download v9.4 Setup.
    Important: Never update Prism Launcher to the latest version.
  2. Install the launcher. Click "Next" until the final screen. Do not add a Microsoft account if prompted.
  3. Close the launcher and open CMD as Administrator.
  4. Copy and paste the following code into CMD and press Enter:
    echo {"accounts": [{"entitlement": {"canPlayMinecraft": true,"ownsMinecraft": true},"type": "MSA"}],"formatVersion": 3} > %appdata%/PrismLauncher/accounts.json
  5. Run Prism Launcher. In the top right corner, click the drop-down menu and select Manage Accounts.
  6. Click Add Offline.
  7. Enter your preferred username and click OK, then close the window.
  8. Click Add Instance. Select Minecraft 1.21.1 and scroll down to select Fabric. Click OK.
  9. Ensure your username is selected in the top right, then hit Launch.
  10. The game will download files. Once it runs, turn off the Narrator and accept the terms.
  11. Close the game. Click the Folder icon (located below Launch) to find the mods folder.
  12. Download and place these 3 files into the mods folder:
    • Simple Voice Chat
    • Sodium
    • Iris Shaders
  13. Hit Launch again.
  14. On the home screen, click Multiplayer -> Add Server.
  15. Name: NITKSMP | Address: NITKSMP.aternos.me:13198
  16. Hit Done and join the server! 🎉

Pro-Tips:

• Press 'V' in-game to setup Voice Chat.
• If the server is offline, ping one of the three admins to start it.
